## [1.0.0] - 2026-05-21

### Added
- Central Monitoring API with install token and API key authentication
- Grafana Alloy node agent with automatic registration
- Port probe monitoring with per-port labels
- Server inventory: hostname, IP, client, account, display name management
- Prometheus remote_write receiver on central server
- Grafana dashboards: All Servers, Server Drill-Down, CPU/Memory/Disks/Network/Ports/Processes
- One-liner node install script (install-alloy.sh)
- Central server setup script (setup-server.sh)
- Multi-client/account org management in Grafana
